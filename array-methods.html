at()

The at() methos takes an interger value and retruns the item at that index, allowing for positive and negative integers. Negative integers count back from the last item in the array.

This is not to suggest there is anything wrong with using the square bracket notation.

For example array[0] would return the first item. However instead of using array.length of latter items; e.g. array[array.length-1] for the last item,

you can call array.at(-1)
